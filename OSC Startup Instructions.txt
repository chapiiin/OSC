Create the following directory: c:\sqlite\db\osc_system.db

To setup the database initially, the line inside of the main() function of the file OSC.py that states "D.setup_database()" will need to be uncommented.
This will populate the database with the initial data.

Navigate to the directory storing the "OSC.py" file from the command line

Run the program with "python osc.py"

Valid initial user accounts include the following username:password combination:
-admin:password
-connor:password
-heather:password
